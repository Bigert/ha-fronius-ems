# ha-fronius-ems
Read and write to Fronius local API och change Soft-Limit and more.

This is a test project created with ChatGPT with the aim of reading soft limit in Fronius Gen24 10kW >firmware 40 or 2026.
I also have a Fronius Symo 15kW and the idea is to use the EMS to limit the total output to grid if spot price is low.

I am not qualified to give support but post this as a stepping stone for others.
